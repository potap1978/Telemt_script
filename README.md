# Telemt MTProxy Manager

Скрипт создан для друзей Телеграм канала [Клуб «На связи»](https://t.me/na_sviazi66)

**Что это?**  
Этот скрипт позволяет быстро развернуть свой MTProxy-сервер для Telegram с защитой от блокировок. Всё что нужно — запустить одну команду, и через пару минут у вас будет готовый прокси с возможностью управления через Telegram бота.

**Зачем это нужно?**  
Telegram часто блокируют на уровне провайдеров. MTProxy с маскировкой под обычный HTTPS-трафик (TLS-fronting) помогает обойти эти ограничения. Скрипт автоматически настраивает всё необходимое и позволяет управлять пользователями прямо из Telegram.

## Возможности

- 🚀 **Автоматическая установка** — компилирует telemt из исходников, настраивает systemd, создаёт пользователя
- 👥 **Управление пользователями** — добавляйте, удаляйте, просматривайте список (каждый получает свою ссылку)
- 🔧 **Смена SNI** — маскируйте трафик под легитимные сайты (google.com, cloudflare.com и другие)
- 🔌 **Смена порта** — по умолчанию 7443, можно изменить на любой
- 🤖 **Telegram бот** — управляйте прокси удалённо: создавайте/удаляйте пользователей, меняйте настройки, смотрите статус
- 📊 **Статус сервера** — IP, порт, SNI, количество пользователей, логи
- 🔒 **Ограничение по IP** — можно настроить, чтобы ссылкой мог пользоваться только один человек одновременно

# Telemt MTProxy Manager

**Скрипт создан для друзей Телеграм канала [Клуб «На связи»](https://t.me/na_sviazi66)**

---

## 📖 Что это?

Этот скрипт позволяет развернуть **полную инфраструктуру MTProxy-сервера** для Telegram с защитой от блокировок и удобным управлением.

Всё, что нужно — запустить **одну команду**, и через несколько минут у вас будет:

- ⚡ Рабочий прокси-сервер
- 🌐 Веб-панель для управления
- 🤖 Telegram бот для удаленного администрирования
- 🔒 HTTPS с корректно работающим WebSocket

---

## 🎯 Зачем это нужно?

Telegram часто блокируют на уровне провайдеров.  
**MTProxy с TLS-маскировкой** (под обычный HTTPS-трафик) помогает обойти эти ограничения.

Скрипт автоматически настраивает всё необходимое и даёт вам удобные инструменты управления.

---

## ✨ Возможности

### 🔧 Основной прокси (telemt)

| Функция | Описание |
|---------|----------|
| 🚀 Автоматическая установка | Компиляция telemt из исходников, настройка systemd, создание системного пользователя |
| 👥 Управление пользователями | Добавление, удаление, просмотр списка. Каждый получает свою уникальную ссылку |
| 🔒 Ограничение по IP | Возможность настроить использование ссылки только с одного IP |
| 🔌 Смена порта | По умолчанию 7443, можно изменить на любой |
| 🎭 Смена SNI | Маскировка трафика под легитимные сайты (google.com, cloudflare.com и другие) |

### 🌐 Web панель

| Функция | Описание |
|---------|----------|
| 📊 Dashboard | Статистика подключений, активные IP, трафик в реальном времени |
| 👥 Управление пользователями | Просмотр, добавление, удаление через удобный веб-интерфейс |
| 🔧 Настройка прокси | Смена порта, SNI, лимитов прямо из панели |
| 🔒 HTTPS поддержка | Работа через Nginx с самоподписным сертификатом |
| 🔌 WebSocket | Корректно настроен, данные обновляются в реальном времени |

### 🤖 Telegram бот

| Функция | Описание |
|---------|----------|
| 📊 Статус сервера | IP, порт, SNI, количество пользователей |
| 👥 Управление пользователями | Создание, удаление, просмотр списка |
| 🔧 Смена настроек | Порт, SNI, лимиты IP |
| 🔄 Перезапуск telemt | Прямо из Telegram |

### 🔄 Общие функции

| Функция | Описание |
|---------|----------|
| 📋 Статус и информация | Полная диагностика системы |
| 🔐 Смена логина/пароля Web панели | Если забыли учётные данные |
| 🗑️ Полное удаление | Чистое удаление всех компонентов |
